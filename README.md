# Vaultary

Vaultary is a blockchain wallet monitoring platform that alerts you when balances change or large transfers occur.

It is designed for:

- DAO treasury monitoring
- crypto wallet security
- whale wallet tracking
- exchange wallet monitoring

## Features

• Multi-chain wallet monitoring  
• Balance change alerts  
• Large transfer alerts  
• Discord / Telegram / Email notifications  
• Organization and team monitoring
